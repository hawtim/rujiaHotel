# rujiaHotel
如家酒店与微信支付合作活动页面

页面开发是在chrome中完成的，之后通过优雅降级的方式逐步针对ie系列浏览器进行测试和修复
页面兼容至ie7，在index.html的页面头部使用meta标签
<code><meta http-equiv="X-UA-Compatible" content="IE=7,chrome=1"></code>

学习之处：更好掌握兼容IE的方式
